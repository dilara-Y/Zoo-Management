# Zoo-Management
A zoo management system built with Java using Object-Oriented Programming principles.

## Features
- Add, update, delete and list animals (CRUD operations)
- Animal and enclosure management
- Clean OOP architecture

## Technologies Used
- Java
- OOP Design Patterns

## How to Run
1. Clone the repository
```
   git clone https://github.com/dilara-Y/zoo-manager.git
```
2. Open with IntelliJ IDEA — File → Open → select the project folder
3. Run `Main.java`
